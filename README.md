🧠 Quiz App (React JS)

The Quiz App is an interactive web application built using React.js that allows users to answer multiple-choice questions and test their knowledge. The application manages quiz state efficiently using the useReducer hook, making it suitable for handling complex state logic.

🚀 Features

❓ Multiple-choice quiz questions

📊 Real-time score calculation

⏭ Navigate between questions

🏁 Display final quiz result

⚡ Efficient state management using useReducer

📱 Responsive user interface

🛠 Tech Stack

React.js

JavaScript (ES6)

HTML5

CSS3

React Hooks (useReducer, useEffect)
